У компонента DataGrid есть API, позволяющий напрямую редактировать записи в ячейках. Во время редактирования ячейки будет отображён UI с кнопками для сохранения и отмены изменений.

Attributes:

  • editorEnabled - определяет включен или нет втроенный редактор DataGrid. Если редактор включен, тогда пользователь может открыть его двойным кликом по строке или нажатием клавиши Enter на активной строке. Редактор может быть открыт программно методом editItem(Object).

  • editorBuffered - отределяет включен ли режим буфферизации.

  • editorSaveCaption - задает заголовок кнопки save в редакторе DataGrid.

  • editorCancelCaption - задает заголовок кнопки cancel в редакторе DataGrid.
