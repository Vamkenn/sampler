The Dialog facets are non-visual components that provide an ability to pre-configure standard dialogs windows in XML descriptor. Dialogs can be pre-configure via corresponding facets:

  • MessageDialogFacet - corresponds to Dialogs#createMessageDialog;
  • OptionDialogFacet - corresponds to Dialogs#createOptionDialog;
  • InputDialogFacet - corresponds to Dialogs#createInputDialog.

Dialogs can be shown explicitly via corresponding method: show() or subscribed to any action or button by id:

  • onButton - specifies the button id that triggers a dialog;
  • onAction - specifies the action id that triggers a dialog.
