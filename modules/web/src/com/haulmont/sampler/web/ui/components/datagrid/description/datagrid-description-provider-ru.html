setCellDescriptionProvider() примает экземпляр CellDescriptionProvider, который будет использоваться для генерации всплывающих подсказок для отдельных ячеек DataGrid

setRowDescriptionProvider() примает экземпляр RowDescriptionProvider, который будет использоваться для генерации всплывающих подсказок для строк DataGrid. Если CellDescriptionProvider так же установлен, то RowDescriptionProvider будет использован только для тех ячеек, для которых CellDescriptionProvider возвращает null.
