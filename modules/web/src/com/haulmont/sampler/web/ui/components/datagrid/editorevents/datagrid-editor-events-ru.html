• EditorOpenListener - срабатывает перед тем, как редактор будет открыт.

  • EditorPreCommitListener - срабатывает перед тем, как изменения элемента отправятся в datasource.

  • EditorPostCommitListener - срабатывает после того, как изменения элемента отправлены в datasource.

  • EditorCloseListener - срабатывает, когда встроенный редактор DataGrid закрылся.
