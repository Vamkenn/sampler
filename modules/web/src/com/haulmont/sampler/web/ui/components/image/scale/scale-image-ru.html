С помощью атрибута ScaleMode можно задать тип масштабирования изображения:

  • NONE - изображение не масштабируется.

  • CONTAIN - изображение сжимается или растягивается до минимального размера компонента при сохранении пропорций.

  • SCALE_DOWN - изображение сравнивает разницу между NONE и CONTAIN, чтобы найти наименьший подходящий размер.

  • FILL - изображение растягивается по размеру компонента.
