The DataGrid component has an API for inline editing of records in the cells. When an item is being edited, the inline edit UI with default save and cancel buttons is displayed.

Attributes:

  • editorEnabled - define whether or not the item editor UI is enabled for this DataGrid. When the editor is enabled, the user can open it by double-clicking a row or hitting enter when a row is focused. The editor can also be opened programmatically using theeditItem(Object) method.

  • editorBuffered - define if the buffered editor mode is active.

  • editorSaveCaption - the caption on the save button in the DataGrid editor.

  • editorCancelCaption - the caption on the cancel button in the DataGrid editor.
