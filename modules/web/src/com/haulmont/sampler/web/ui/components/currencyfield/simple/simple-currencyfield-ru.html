CurrencyField - компонент для просмотра и редактирования валютных значений. Он состоит из поля ввода текста и надписи и текстовой записи валюты. Данную запись можно настроить с помощью следующих методов:

  • currency - текстовое значение надписи.

  • showCurrencyLabel - отображать/не отображать данную надпись. По умолчанию надпись отображается.

  • currencyLabelPosition - устанавливает положение надписи (справа или слева). По умолчанию - справа.
