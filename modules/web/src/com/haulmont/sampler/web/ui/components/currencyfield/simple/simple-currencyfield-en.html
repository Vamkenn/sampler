CurrencyField - component for representing and editing currency values. It consists of text input field and (optionally) currency label.

CurrencyField's label can be changed by the following methods:

  • currency - a text that will be a content of the currency label.

  • showCurrencyLabel - enables or disables currency label displaying. This option is enabled by default.

  • currencyLabelPosition - sets where the currency label should be. CurrencyLabel.RIGHT - default value.
