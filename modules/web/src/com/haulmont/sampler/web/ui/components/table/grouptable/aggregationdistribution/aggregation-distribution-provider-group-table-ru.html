AggregationDistributionProvider позволяет произвести изменения в данных, когда агрегированное значение было изменено вручную.

Данная фунциональность поддерживается только при aggregationStyle="TOP" режиме. Чтобы обработчик работал нужно разрешить редактирование агрегации в колонке <aggregation editable="true">.
