Screen facets - это невизуальные компоненты, позволяющие декларативно настроить открытие экранов и передачу параметров в экраны в XML дескрипторе экрана. Открытие экранов может быть настроено с помощью следующих невизуальных компонентов:

  • ScreenFacet - соответствует вызову ScreenBuilders#screen;
  • EditorScreenFacet - соответствует вызову ScreenBuilders#editor;
  • LookupScreenFacet - соответствует вызову ScreenBuilders#lookup.

Экраны могут быть открыты с помощью с помощью вызова соответсвующего метода: show() или с помощью подписки на действие или кнопку по идентификатору:

  • onButton - определяет идентификатор кнопки, которая вызовет открытие экрана;
  • onAction - определяет идентификатор действия, которое вызовет открытие экрана.
