The Screen facets are non-visual components that provide an ability to pre-configure screen opening and pass properties in XML descriptor. Screens can be pre-configure via corresponding facets:

  • ScreenFacet - corresponds to ScreenBuilders#screen;
  • EditorScreenFacet - corresponds to ScreenBuilders#editor;
  • LookupScreenFacet - corresponds to ScreenBuilders#lookup.

Screens can be opened explicitly via corresponding method: show() or subscribed to any action or button by id:

  • onButton - specifies the button id that opens a screen;
  • onAction - specifies the action id that opens a screen.
